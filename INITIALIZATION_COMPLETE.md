- 使用 Python 的 `http.server` 模块启动了本地服务器
- 服务器地址: `http://localhost:8000`
- 可以通过此地址访问应用程序

## 使用方法

1. 在浏览器中打开 `http://localhost:8000`
2. 输入要转换的文本
3. 输入提示音频的 URL（或使用默认提供的 URL）
4. （可选）输入提示文本
5. 点击 "生成语音" 按钮
6. 等待生成完成后，使用音频播放器播放生成的语音

## 注意事项

- 项目使用的是 Gitee AI 的 TTS API，需要有效的 API 密钥
- 提示音频 URL 必须是可公开访问的
- 推荐使用 .wav 格式的提示音频
- 如果遇到问题，请检查浏览器控制台的错误信息
