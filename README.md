# 语音生成项目

这是一个基于 Web 的语音生成应用，使用 Gitee AI 的 TTS (Text-to-Speech) API 实现。

## 功能特点

- 文本转语音功能
- 支持通过 URL 提供提示音频来克隆特定音色
- 支持添加提示文本（可选）
- 响应式设计，适配各种屏幕尺寸
- 本地存储上次使用的提示音频 URL

## 项目结构

```
kelong-audio-main/
├── index.html          # 主页面文件
├── api.mc             # API 密钥配置
├── kelong/            # 示例音频目录
│   └── 克隆音色.MP3    # 示例克隆音色音频
└── README.md          # 项目说明文档
```

## 使用方法

1. 克隆或下载项目到本地
2. 确保您有有效的 Gitee AI API 密钥（已在 `api.mc` 文件中配置）
3. 在浏览器中打开 `index.html` 文件
4. 输入要转换的文本
5. 输入提示音频的 URL（支持公开可访问的 .wav 格式音频）
6. （可选）输入提示音频对应的文本
7. 点击 "生成语音" 按钮
8. 等待生成完成后，使用音频播放器播放生成的语音

## 注意事项

- 提示音频 URL 必须是可公开访问的
- 推荐使用 .wav 格式的提示音频
- 音频生成可能需要一些时间，取决于网络状况和文本长度
- 如果遇到 CORS 跨域错误，请检查浏览器控制台的网络请求

## 技术栈

- HTML5
- Tailwind CSS
- JavaScript
- Gitee AI TTS API

## API 配置

API 密钥存储在 `.env` 文件中：
```
API_KEY=您的API密钥
```

注意：在生产环境中，建议不要将 API 密钥直接暴露在前端代码中，应使用后端代理服务。
